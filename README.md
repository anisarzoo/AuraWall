# AuraWall | Premium Generative Wallpaper Studio

AuraWall is a high-performance, browser-based generative art studio that creates unique wallpapers for any screen size. It's built for those who want a fresh, artistic background every day without relying on external image APIs.

## ✨ Features

- **Living Wallpapers**: Every pattern features smooth 60FPS animations.
- **Interactive Parallax**: Generative elements react to your mouse movement.
- **Advanced Studio**: Real-time sliders for Complexity and Hue Variation.
- **Four Art Styles**:
  - **Aurora Drift**: Fluid, glowing northern lights.
  - **Geometric Echo**: Mathematical grids and lines.
  - **Nebula Mist**: Deep space interstellar gas clouds.
  - **Silk Minimal**: Clean, ethereal bezier curves.
- **High-Resolution Export**: Download your creations as high-quality PNGs.
- **Local First**: Zero external dependencies, works completely offline.

## 🚀 Getting Started

1. Clone this repository or download the source code.
2. Open `index.html` in any modern web browser.
3. Use the studio panel (gear icon) to customize your art.
4. Click the download icon to save your new wallpaper.

## 🛠️ Built With

- **HTML5 Canvas**: Pure generative rendering.
- **Vanilla JavaScript**: High-performance animation loops.
- **Modern CSS**: Glassmorphism and responsive layout.
